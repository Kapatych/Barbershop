## Features

+ Adaptive Web Design (320px, 768px, 1200px)
+ BEM 
+ SASS
+ Gulp 4
+ Cross-browser compatibility (last versions)
+ Accessibility
+ Custom JS slider 
+ Crop images for different screens (with <picture>)
+ Valid HTML/CSS W3C
